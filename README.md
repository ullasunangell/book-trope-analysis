# Book Trope Analysis

This project explores how different story elements (tropes) relate to reader engagement and ratings using Goodreads book data.

The goal of this project is to demonstrate data analysis skills while exploring patterns in publishing and reader preferences.

## Dataset

Source: Goodreads Books Dataset

The dataset contains information about 700+ books including:

- Title
- Author
- Rating
- Number of ratings
- Description
- Genres
- Publication date
- Page count

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Analysis Steps

1. Data loading and exploration
2. Data cleaning
3. Rating distribution analysis
4. Engagement analysis (ratings count)
5. Correlation analysis (pages vs rating)
6. Trope detection using book descriptions
7. Visualisation of trope frequency and engagement

## Key Findings

- Goodreads ratings cluster between **3.7 and 4.2**, showing typical rating inflation.
- Book popularity is highly skewed: a small number of books receive millions of ratings.
- Book length has **very little correlation with rating**.
- Certain narrative elements appear to correlate with higher reader engagement.
